Il est préconisé de déclarer le titre de page avec un <h1> et de suivre ensuite une hiérarchie de titres logique.

Des styles de surcharge sont proposés nativement dans la stack pour modifier la taille du texte (zt = zone de travail) :

  • Style «bl-zt-h1» : 1.5 rem
  • Style «bl-zt-h2» : 1.3 rem
  • Style «bl-zt-h3» : 1.25 rem
  • Style «bl-zt-h4» : 1.2 rem
  • Style «bl-zt-h5» : 1.1 rem
  • Style «bl-zt-h6» : 1 rem

Il est préconisé d’utiliser les surcharges de styles suivants pour les applications desktop utilisant la stack :

<H1 class= «bl-zt-h1» …/>

<H2 class=«bl-zt-h2» …/>

<H3 class=«bl-zt-h3» …/>

<H4 class=«bl-zt-h4» …/>

<H5 class=«bl-zt-h5» …/>
<H6 class=«bl-zt-h6» …/>
